# Data Report Workflow

Multi-skill orchestration for generating EEA data reports.

## Skills Used

1. **chart** → Generate visualizations
2. **xlsx** → Format tabular data
3. **doc** → Compile final report

## Workflow

```
┌─────────┐    ┌─────────┐    ┌─────────┐
│  chart  │ -> │  xlsx   │ -> │   doc   │
└─────────┘    └─────────┘    └─────────┘
```

## Trigger

"Generate a data report for [topic]"

Workflow definition to be expanded as skills are added.
