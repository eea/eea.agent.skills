# EEA Glossary

Common terms, acronyms, and concepts used across EEA projects. Agents should use these definitions consistently.

---

|---------|-----------|---------|
| EEA | European Environment Agency | The organization |
| EIONET | European Environment Information and Observation Network | Data network |
| CDR | Copernicus Data Repository | Satellite data |
| ETC | European Topic Centre | Thematic centers |
| NFP | National Focal Point | Country representatives |
| NRC | National Reference Centre | Expert centers |
| SOER | State of the Environment Report | Flagship report |
| AQR | Air Quality Report | Annual report |
| LRTAP | Long-Range Transboundary Air Pollution | Convention |
| WFD | Water Framework Directive | EU legislation |
| MSFD | Marine Strategy Framework Directive | EU legislation |

## Data Formats

| Format | Use | Notes |
|--------|-----|-------|
| **GML** | Geographic Markup Language | ISO 19136 compliant |
| **GeoJSON** | Geographic JSON | Preferred for web APIs |
| **NetCDF** | Network Common Data Form | Climate/meteorological data |
| **INSPIRE** | EU spatial data standard | Mandatory for geospatial data |
| **SDMX** | Statistical Data and Metadata eXchange | Used for statistical reporting |

## Writing Style

When referring to EEA in text:
- **First mention:** "European Environment Agency (EEA)"
- **Subsequent mentions:** "EEA" or "the Agency"
- **Never:** "the EEA" (redundant)

When referring to EEA products:
- Use official product names (e.g., "State of the Environment Report", not "SOER report")
- Include version numbers where relevant (e.g., "SOER 2020")
