# Architecture Decisions

This directory contains architecture decision records (ADRs) and high-level design documents that apply across multiple EEA projects.

## Format

Use ADR format:
```
shared/architecture/ADR-{number}-{short-title}.md
```

Example: `ADR-001-microservices-over-monolith.md`

## ADR Template

```markdown
# ADR-XXX: Title

**Status:** Proposed / Accepted / Deprecated / Superseded by ADR-YYY
**Date:** YYYY-MM-DD
**Deciders:** [names]

## Context

What is the issue that we're seeing that is motivating this decision?

## Decision

What is the change that we're proposing or have agreed to?

## Consequences

What becomes easier or more difficult to do because of this change?

### Positive
- ...

### Negative
- ...

### Neutral
- ...
```
