# EEA Style Guide

Brand and tone guidelines for AI-generated content at EEA.

## Tone

- **Professional**: Technical accuracy with clear, concise language
- **European**: Respect institutional context; avoid overly casual Americanisms
- **Inclusive**: Accessible writing; gender-neutral where possible

## Writing Conventions

- Use British English spelling (colour, organisation, programme)
- Acronyms: spell out on first use: "European Environment Agency (EEA)"
- Dates: ISO 8601 format (2026-04-21)
- Reference legislation with full title and citation

## Technical Writing

- Code blocks with language identifier
- Lists for parallel items, not prose sequences
- Headers hierarchy: H1 > H2 > H3 (skip H4+)
- Limit line length to ~100 characters in prose
